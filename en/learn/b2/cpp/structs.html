Data Structures with struct

In C++, a structure (struct) lets you group different pieces of data together under one name. You can think of it like a custom data type that you design yourself.

What Is a struct?

A struct groups related variables (called members) of possibly different types.

Syntax

struct TypeName {
    member_type1 member_name1;
    member_type2 member_name2;
    // ...
};
  • TypeName → the name of your new type
  • Inside { } → the members (variables) that belong to this type

Example: A product Structure

#include <iostream>
using namespace std;

struct product {
    int weight;
    double price;
};

int main() {
    product apple;
    product banana, melon;

    apple.weight = 150;      // grams
    apple.price  = 1.99;     // riyals

    banana.weight = 120;
    banana.price  = 0.99;

    cout << "Apple: "  << apple.weight  << "g, $" << apple.price  << endl;
    cout << "Banana: " << banana.weight << "g, $" << banana.price << endl;
}

Accessing Members

Use the dot operator .:

  • apple.weight → int
  • apple.price → double

You can read/write them just like normal variables.

Declaring Objects Right After the struct

You can declare variables at the end of the struct definition:

struct product {
    int weight;
    double price;
} apple, banana, melon;

Here:

  • product is the type name
  • apple, banana, melon are variables (objects) of that type

Pointers to Structures and the -> Operator

Just like int* can point to an int, a product* can point to a product.

#include <iostream>
#include <string>
using namespace std;

struct product {
    int weight;
    double price;
};

int main() {
    product aproduct;
    product* pproduct = &aproduct;   // pointer to aproduct

    string temp;

    cout << "Enter weight: ";
    getline(cin, temp);
    pproduct->weight = stoi(temp);

    cout << "Enter price: ";
    getline(cin, temp);
    pmovie->price = stod(temp);

    cout << "\nYou have entered:\n";
    cout << pproduct->weight << " (" << pmovie->price << ")\n";
}

Dot vs Arrow

Expression Meaning
obj.member Member of the object obj
ptr->member Member of the object pointed to by ptr
(*ptr).member Same as ptr->member

Nested Structures

A member of a structure can itself be another structure.

#include <iostream>
#include <string>
using namespace std;

struct movies_t {
    string title;
    int year;
};

struct friends_t {
    string name;
    string email;
    movies_t favorite_movie;
};

int main() {
    friends_t charlie;

    charlie.name = "Charlie";
    charlie.email = "charlie@example.com";
    charlie.favorite_movie.title = "The Matrix";
    charlie.favorite_movie.year  = 1999;

    cout << charlie.name << "'s favorite movie is "
         << charlie.favorite_movie.title
         << " (" << charlie.favorite_movie.year << ")\n";
}

You can also use a pointer to friends_t:

friends_t charlie;
friends_t* pfriends = &charlie;

pfriends->favorite_movie.title = "Inception";
pfriends->favorite_movie.year  = 2010;

Here:

  • pfriends->favorite_movie is a movies_t
  • pfriends->favorite_movie.title is a string
  • You can chain . and -> depending on what you have (object vs pointer).
