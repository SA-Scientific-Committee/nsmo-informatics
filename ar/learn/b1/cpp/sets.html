المجموعات (Sets)

set في C++ هو حاوية لعناصر من نفس النوع ويحقق الخصائص التالية:

  • يحتوي على عناصر فريدة فقط
  • يكون دائمًا مرتبًا
  • يسمح بالإدراج والحذف والاستعلام بكفاءة
Caution

يمكن أن يحتوي set فقط على أنواع يمكن مقارنتها باستخدام معامل <.

الإعلان

#include <iostream>
#include <set>

using namespace std;

int main () {

    set<int> my_int_set;

    set<double> my_double_set;

    set<string> my_string_set = {"abc", "def", "ghi"};

}

الحجم (Size)

تعيد الدالة size عدد العناصر في الحاوية.

set<double> s = {1, 2, 3.2, 32.0 / 10.0};

cout << s.size(); // الناتج 3 لأن 3.1 = 31.0/10.0

الإدراج (Insert)

تضيف الدالة insert عنصرًا إلى الحاوية فقط إذا لم يكن موجودًا مسبقًا وتحافظ على ترتيب العناصر في المجموعة.

set<int> s = {1, 3};

s.insert(2); // الآن s = {1, 2, 3} // ليس {1, 3, 2}

s.insert(3); // لا يتغير s

الحذف (Erase)

تزيل الدالة erase عنصرًا من الحاوية فقط إذا كان موجودًا مسبقًا.

set<string> s = {"abc", "def"};

s.erase("xyz"); // لا يتغير s لأنه لا يحتوي على "xyz"

s.erase("abc def"); // لا يتغير s لنفس السبب

s.erase("abc"); // الآن s = {"def"}
Noneتحدي (Challenge)

باستخدام الدوال المذكورة أعلاه فقط، هل يمكنك التحقق مما إذا كانت المجموعة تحتوي على قيمة محددة؟

Tipالحل (Solution)

أولًا، سنتحقق من حجم المجموعة، لنسمه \(S_1\). بعد ذلك، سندرج هذه القيمة في المجموعة ونتحقق من الحجم مرة أخرى، لنسم الحجم الجديد \(S_2\). إذا كان \(S_1 = S_2\) فهذا يعني أن المجموعة تحتوي على القيمة، وإلا فهي لم تحتوي عليها ويجب إزالتها.

العد (Count)

تعيد الدالة count عدد مرات ظهور القيمة في الحاوية. بما أن جميع العناصر مميزة فإنها تعيد 1 إذا كانت موجودة و 0 إذا لم تكن موجودة.

set<int> s = {1, 2, 3};

cout << s.count(2) << '\n'; // الناتج: 1
s.insert(2);
cout << s.count(2) << '\n'; // الناتج: 1 // لأن المجموعات لا تحتوي على تكرارات
s.erase(2);
cout << s.count(2) << '\n'; // الناتج: 0
Warning

على الرغم من أن set له ترتيب محدد، لا يمكننا الوصول إلى عنصر محدد باستخدام معامل [].

تابع القراءة لمعرفة كيفية الوصول إلى العناصر في set.

التكرار (Looping)

للتكرار على قيم set نستخدم حلقة for بنطاق (range-based for loop) كما يلي:

set<string> my_string_set = {"abc", "def", "ghi"};

for (string s : my_string_set) {
    cout << s << '\n';
}
الناتج:
abc
def
ghi
Noneتحدي (Challenge)

ما هو ناتج البرنامج التالي:

set<int> s = {3, 2, 1};

for (int i : s) {
    cout << i << ' ';
}
Tipالحل (Solution)

الناتج هو 1 2 3 لأن sets تكون دائمًا مرتبة.
